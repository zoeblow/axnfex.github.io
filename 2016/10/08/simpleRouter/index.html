以下方法执行完毕后均返回实例本身。

参数 取值类型 默认值 描述
container String ‘#container’ container 容器的选择器
url String 以 / 开头的 url，会体现在 hash，支持参数，如：/user/:userId
className String 可选，该页面可以添加的额外类名，以便控制该页面下的样式
render Function 页面渲染方法，支持同步和异步, 可以直接返回 html 字符串，可以返回 <template> 标签内容
bind Function 执行绑定事件的方法,this 指向当前页面容器

由于每个API都灰常灰常的简单，我就不一一分析每一个的用法了.
