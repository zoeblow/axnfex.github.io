发表于 2016-09-13 by 阿牛   |   分类于 git ， github   |  

git连接github远程仓库方式有2种，https和ssh，https的需要输入帐号和密码，所以提交代码比较繁琐，而ssh是在提交的时候向远程服务器发送本地私钥，然后服务器用公钥与之配对验证，这种方式非常方便，但是需要做一些前期工作。
