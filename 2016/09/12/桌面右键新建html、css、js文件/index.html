桌面右键新建html、css、js文件

发表于 2016-09-12 by 阿牛   |   分类于 技术分享   |  

win+r打开运行菜单，输入regedit，确定


展开名为HKEY_CLASSES_ROOT项（第一个）



下拉滚动条，找到想要新建文件的拓展名项，右击新建项，命名为ShellNew



右击ShellNew项，新建字符串值，命名为FileName



双击右边新建的FileName，数值数据填写为模版文件路径，点击确定，等几秒后就可以在桌面右键新建中看到添加的快捷方式了。
