制作iconfont图标

发表于 2016-12-25 by 阿牛   |   分类于 技术分享   |  

网站页面上会用到各种小图标，前端以前制作页面的时候，会将这些图片一个个切下来，通过标签设置backgroud使用，这样做会导致请求过多，造成性能瓶颈。后来有人就发明了css sprite(雪碧图)，将这些图标合并到一个图片上，然后通过backgroud-position定位使用，好处是显而易见的，请求瞬间就降低了，图片集中起来也方便维护。

Atom编辑器使用markdown走过的坑

发表于 2016-12-25 by 阿牛   |   分类于 踩坑记录   |  

Atom仅仅提供markdown预览功能，无法满足日常的写作操作，需安装第三方插件
1.markdown-scroll-sync 实现同步滚动
2.markdown-writer 实现新建文章、上传图片、代码格式化等功能
由于要配合hexo写作使用，因此需修改配置：
File Extension值修改为：.md
Site Engine值修改为：hexo
Site Drafts Directory值修改为：{title}
Site Posts Directory值修改为：/
修改上面2个的原因是由于使用了hexo-asset-image这个插件，方便使用本地图片，不过这个插件也有约束的地方就是图片文件夹和.md文件同目录，并且文件夹名称和文章title相同
